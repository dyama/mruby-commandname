# mruby-commandname

Get self command path string for embedded/precompiled mruby code in executable binary.

Please use it at **your own risk**.

## Example

On linux:

    command_name
     => "/home/dyama/mruby/bin/mirb"
 
On Windows:

    command_name
     => "e:\\mruby\\bin\\mirb.exe"
